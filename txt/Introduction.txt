Introduction
============

BISON is a bisulfite-converted short-read aligner that can natively utilize
high-performance computing clusters to increase speed. This brief tutorial
provides instructions for installation as well as usage using a number of
example datasets that accompany this document.

The version of the tutorial that you're reading has been produced for version
0.3.3 of BISON.

If you have questions not covered by this document or notice errors, please feel
free to post a message http://seqanswers.com/forums/showthread.php?t=31314[here].
